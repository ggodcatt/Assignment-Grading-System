# 📝 Assignment Grading System 🗂️

Assignment grading system เป็นโปรเจคกลุ่มที่มีสมาชิกทั้งหมด 7 คน โดยกลุ่มของเราได้รับโจทย์จากลูกค้าว่าให้สร้างเว็บไซต์ระบบการให้คะแนนการบ้าน ที่ช่วยให้อาจารย์สามารถตรวจและประกาศคะแนนได้ในระบบเดียวกัน โดยอาจารย์สามารถสร้าง Assignment สร้าง Rubric สำหรับตรวจ และสามารถเลือกตรวจเป็นรายข้อหรือรายบุคคลได้ ในส่วนของนักศึกษาสามารถเข้าสู่ระบบเพื่อตรวจสอบคะแนนของแต่ละ Assignment ในแต่ละ Course ได้
